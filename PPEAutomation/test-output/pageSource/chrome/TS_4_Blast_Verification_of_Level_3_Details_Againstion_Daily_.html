Variance

This feature allows you to compare a single branch, multiple branches, or an entire region’s performance versus the prior day, month, or year to determine trends in revenue, expenses and production.
A financial month is not considered reconciled and/or closed until the 15th of the following month:
- If the month-over-month variance is run on the 1st-14th, BLAST will compare YTD 3 months prior to YTD 2 months prior.

Variance

This feature allows you to compare a single branch, multiple branches, or an entire region’s performance versus the prior day, month, or year to determine trends in revenue, expenses and production.
A financial month is not considered reconciled and/or closed until the 15th of the following month:
- If the year-over-year variance is run on the 1st-14th, BLAST will compare YTD 2 months prior over the same month span for the prior year.
- If the year-over-year variance is run on the 15th-31st, BLAST will compare YTD 1 month prior over the same month span for the prior year.

How to interpret the formatting:

- If a line item is Revenue generating, positive variance amounts will be presented in green to indicate an increase in revenue, while negative variance amounts will be presented in red to indicate a decrease in revenue.
- If a line item is represents an Expense, positive variance amounts will be presented in red to indicate an increase in expense, while negative variance amounts will be presented in green to indicate a decrease in expense.
